## 快速创建项目脚手架

> 目前支持创建`antd-admin`中后台管理项目，持续更新中。

### 安装

```shell
  npm i -g qwd-cli
```

### 更新

```shell
  npm update qwd-cli
```

### 创建项目

```shell
  qwd create

  # 选择项目
  ? 请选择项目模板? (Use arrow keys)
  ❯ antd-react-template
  react-native-template
```
